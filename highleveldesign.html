High-Level Design

Module: Core Platform Infrastructure

Combining the individual modules together renders a fully functioning software system for the Online Video Streaming Platform.

Design Element DE-0: System Backbone and Core Services

The basic workflow of the system infrastructure is as follows:

System Component Function/Role
Authentication Service Performs user authentication during sign-in, using Single Sign-On and secure access control.
API Gateway Handles user requests and routes them to the appropriate backend services for processing.
Backend Services Developed using Spring Boot/Java, they process requests and interact with services like video streaming, media processing, and user management.
Load Balancing Distributes incoming traffic efficiently across multiple AWS EC2 instances, ensuring seamless performance.
Auto Scaling Automatically adjusts resources (such as adding/removing EC2 instances) based on system load to handle varying traffic volumes.
Monitoring (AWS) Monitors the system performance, raises alerts, and provides insights into system health using AWS monitoring tools (e.g., CloudWatch).
Data Management Handled by AWS RDS with MySQL, ensuring secure and reliable storage of user, video, and interaction data.
CI/CD Pipelines Facilitates continuous integration and deployment of updates using AWS CodeCommit, CodePipeline, CodeBuild, and CodeDeploy.
